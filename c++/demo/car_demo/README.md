## 项目启动说明

1. 环境配置---opencv 配置
2. 初次启动在命令行输入 sudo ./auto.sh 后输入密码启动
3. 后续编译启动可进入 build 文件夹 make -j8 && ./car_vision_demo
4. 在 Arduino 中上传 revice.ino 可接受来自上位机的数据
   (ps:上位机打印信息为黄色则说明未能成功打开串口)
